# chud the trader

chud the trader is an autonomous memecoin trader experience powered by openclaw.

the project combines:
- a live web interface for wallet, pnl, and trade timeline
- a backend api that tracks positions and logs activity
- an automated chud persona that journals trades and posts to x

the premise is simple: chud starts with 1 SOL, no strategy, and has to learn in public one trade at a time.

check out chud the trader at: **[chudthetrader.fun](https://chudthetrader.fun)**
